# Electric-Vehicle-Charge-and-Range-Analysis

## 📌 Project Overview

This project focuses on analyzing Electric Vehicle (EV) data using data visualization techniques. The aim of this project is to study and compare different EV models based on their battery capacity, driving range, charging time, charging type, and price.

An interactive dashboard was created using Tableau Public to visualize the dataset and generate useful insights about electric vehicle performance and charging characteristics.

---

## 🛠 Tools & Technologies Used

* Tableau Public
* CSV Dataset
* Data Visualization Techniques
* GitHub for project hosting

---

## 📊 Dataset Description

The dataset used in this project contains information about different electric vehicles and their specifications.

The dataset includes the following attributes:

* **EV Model** – Name of the electric vehicle
* **Battery Capacity (kWh)** – Battery capacity of the EV
* **Range (km)** – Distance an EV can travel on a single charge
* **Charging Time (hrs)** – Time required to fully charge the battery
* **Charging Type** – Type of charging used (Fast / Normal)
* **Price (INR)** – Price of the electric vehicle

## 🔍 Key Insights

* Electric vehicles with larger battery capacities generally provide longer driving ranges.
* Fast charging technology significantly reduces charging time.
* EV price varies depending on battery capacity and vehicle performance.
* Range and battery capacity are closely related in determining EV efficiency.

## 🎯 Conclusion

Electric vehicles are becoming increasingly popular due to advancements in battery technology and environmental concerns. This project helps analyze EV performance metrics and understand the relationship between battery capacity, range, and charging characteristics.
